Fix Git Push on Linux Mint

Step-by-step commands to get your project live on GitHub Pages.

  Step 1 — Check your remote & branch
git remote -v
# Should show: origin  https://github.com/YOUR_USERNAME/neuralhire.git
git branch
# Note: is it "main" or "master"?
  Step 2 — Set correct remote URL
git remote set-url origin https://github.com/YOUR_USERNAME/neuralhire.git
  Step 3 — Create GitHub Personal Access Token
# Go to: GitHub → Settings → Developer Settings
# → Personal Access Tokens → Tokens (classic)
# → Generate new token → check ✅ repo → Copy token

git config --global credential.helper store
# This saves your token so you only enter it once
  Step 4 — Commit & Push
git add .
git commit -m "v4.0: PDF resume upload, mock interview, session history"
git push origin main
# Username: your GitHub username
# Password: paste the Personal Access Token (NOT your password)
  Step 5 — Enable GitHub Pages
# GitHub → your repo → Settings → Pages
# Source: Deploy from a branch
# Branch: main / root → Save
# Your site: https://YOUR_USERNAME.github.io/neuralhire/
FAQ

Common Questions

Is it really free? +
Yes, 100% free forever. Google Gemini's free API tier gives 15 requests/minute — more than enough for any interview. No credit card required.
Can the interviewer detect it? +
No. Nothing runs on your interview computer. Your phone is completely separate — screen sharing, task managers, and proctoring software can't access it.
How does PDF resume upload work? +
The app reads your PDF locally in the browser using PDF.js and extracts the text. Your resume never leaves your device — it's processed entirely client-side before being sent to Gemini with your question.
How does mock interview scoring work? +
The AI evaluates your typed or recorded answer across three dimensions: relevance to the question, specificity (metrics/examples), and STAR structure adherence. Scores are 1–10 with written feedback.
Where is my session history stored? +
All session history is stored locally on your phone using localStorage. Nothing is sent to any server. You can clear it anytime from the History tab.
Is my data safe? +
Resume, API key, and history are all stored locally on your phone. Audio is processed through Google's encrypted Gemini API. We have zero access to your data.
Which phones work? +
Any modern smartphone — iPhone (Safari), Android (Chrome), or tablets. Microphone access required for live mode.
